HTML

  • Hypertext Markup Language, the coding language used to structure webpages.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1,h2,h3,h4,h5, and h6 are elements represent the level of headings. Like titles and subtitles to organize a page. it is important to use heading tags in order from h1 to h6 to make the page accessible for screen readers or other devices. The biggest heading h1 should only appear once on the page.
  • The p element represents a psrsgraph or block of text.
  • The ul, ol, and il elements represent Unordered List, Ordered List and List Items.
  • The img element contains info about images that are displayed on the webpage. The src attribute defines the location of the image or source. The alt attribute contains a text string that describes the appearance and functionality of the image (for screen reader and search engine)
  • The br element create a line of empty space, or a line break, between two blocks of content.
  • The a element (anchor) creates links to webpage.
  • The href attribute points to URL for the link.
  • The header appears at the top of the page and usually contains a navigation menu and/or introductory content.
  • The footer appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.
  • The section elements are semantic elements that define a section in a document that contains thematic content.
  • Each section element has an id attribute used in HTML as a way to identify elements and select them.
  • The link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked one.
  • The class attribute I will learn later.

CSS

  • Cascading Style Sheets, which allows you to make your webpages engaging and visually dynamic.
  • Inline CSS - apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page alement ONLY.
  • Internal CSS - contains CSS rules for the webpage in the head section of the HTML file.
  • External CSS - in a separate file. For using an external style sheet we link the file using a "link" element placed in the HTML's head.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * - wild card symbol in the rule, we declare that ALL of the elements on the page will have specified properties.
  • We can combine two selectors into a single CSS rule, using comma.
  • ".card" - a class selector. Any HTML document assignet to the class "card" will gain the properties assignment from CSS.
  • We may choose any color, using # code from HTML Color Codes website

Git

  • A popular version control system that connects to GitHub.com, where you can host and share your code.
  • git status: checks what branch we are currently on.
  • git checkout main: switch to the main branch.
  • git pull origin main: pull down the latest main branch.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript handle logic and allows users to interract with website.
  • A variable is a named container that allows us to store data in our code (string, number, boolean, array, or object).
  • Control flow is the order in which a computer executes code in a script.
  • JavaScrip is case sensitive!
  • Operators are math symbols that produse a result based on two values.
  • Conditionals "if...else" used to test if an expression reterns truth or not.
  • Arrays is a simple variable that is used to hold a group of data, and can store different types of data.
  • To make a loop (for) we should determine the starting point (x=0) for our loop, the condition (x<"length"), and the final statement (x++) wich allows to iterate over each time. Add code we want to be executed for each loop.
  • 2 requirements for a JS function: Define the function & Call the function.
