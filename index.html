HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1,h2,h3,h4,h5, and h6 are elements represent the level of headings. Like titles and subtitles to organize a page. it is important to use heading tags in order from h1 to h6 to make the page accessible for screen readers or other devices. The biggest heading h1 should only appear once on the page.
  • The p element represents a psrsgraph or block of text.
  • The ul, ol, and il elements represent Unordered List, Ordered List and List Items.
  • The img element contains info about images that are displayed on the webpage. The src attribute defines the location of the image or source. The alt attribute contains a text string that describes the appearance and functionality of the image (for screen reader and search engine)
  • The br element create a line of empty space, or a line break, between two blocks of content.
  • The a element (anchor) creates links to webpage.
  • The href attribute points to URL for the link.
  • The header appears at the top of the page and usually contains a navigation menu and/or introductory content.
  • The footer appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.
  • The section elements are semantic elements that define a section in a document that contains thematic content.
  • Each section element has an id attribute used in HTML as a way to identify elements and select them.
  • The class attribute I will learn later.
